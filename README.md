# QwenPaw Android APP

功能丰富的 QwenPaw Agent Android 客户端

## 功能
- 实时对话(SSE流式)
- 多Agent切换
- 文件管理
- 数据统计
- 定时任务
- 主题系统(浅色/深色+6强调色)
- 离线缓存
- 全局搜索
- 会话管理
